# BotC

Starter workspace for the `BotC` project.

## Purpose

This repository is set up as a clean local starting point so we can build the project incrementally and publish it to GitHub when ready.

## Project Structure

- `src/` for application code
- `docs/` for notes, plans, and design decisions
- `assets/` for images, audio, or other static resources
